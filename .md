DevExperience — what is it?
A small **social network for developers**: profiles, posts, and comments.

## What can you do?

- Sign up and log in
- Create and edit **your own profile**
- Browse **all profiles** and open **any single profile**
- Add **work experience** and **education**
- **Posts**, **comments**, **likes / dislikes**

## Tech stack

| Layer  | Stack                     |
| ------ | ------------------------- |
| Server | Node.js, Express          |
| Client | React, Redux, Redux Thunk |
| Data   | MongoDB (Atlas)           |
| UI     | Bootstrap                 |

---

## App structure

- **~12 screens** (landing, auth, profiles, dashboard, posts, etc.)
- **20+ components**, including shared form field helpers

---

## Redux

- **auth** — signed-in or not, user payload
- **profile** — one profile + list of profiles
- **post** — post list + current post
- **errors** — API / validation errors
